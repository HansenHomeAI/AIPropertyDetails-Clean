Help & Documentation

Getting Started

AIPropertyDetails uses OpenAI's o4-mini model to analyze property documents and extract boundary coordinates. Here's how to use it:

  1. Upload a Document: Select the document type and upload your parcel map, plat, or survey document
  2. AI Analysis: The system analyzes your document using advanced visual reasoning
  3. Review Results: View extracted coordinates and property details
  4. Export Data: Download results in JSON, CSV, or KML format
Supported Document Types
Parcel Maps

Property boundary maps showing lot lines, dimensions, and coordinates.

Plat Maps

Subdivision maps with lot and block numbers, street layouts.

Survey Documents

Professional land surveys with precise measurements and coordinates.

API Documentation

You can integrate AIPropertyDetails into your own applications using our REST API:

Upload Document
POST /api/upload

Upload a property document for processing.

Analyze Document
POST /api/analyze

Analyze an uploaded document and extract coordinates.

Validate Coordinates
POST /api/validate

Validate extracted coordinates for accuracy.

Export Results
POST /api/export

Export analysis results in various formats.

Important Notes
Disclaimer

This system is designed to assist with property analysis but should not be used as the sole source for legal or official property boundary determinations. Always consult with licensed surveyors and legal professionals for official property information.
